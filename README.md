Phase 1 Project Guidelines
Learning Goals:

Design and architect features across a frontend
Communicate and collaborate in a technical environment
Integrate JavaScript and an external API
Debug issues in small- to medium-sized projects
Build and iterate on a project MVP
Introduction:
This project involves building a Single Page Application (SPA) using HTML, CSS, and JavaScript. It will interact with a public API, focusing on asynchronous data retrieval with JSON. The entire app must run on a single page, with no redirects, and include interactivity through event listeners.

Project Requirements:

HTML/CSS/JS/Bootstrap or Tailwind frontend
Asynchronous data retrieval from a public API using JSON
Incorporation of at least 3 event listeners
Interactivity, which may include actions like "like" buttons or comments
Adherence to good coding practices (DRY code)
